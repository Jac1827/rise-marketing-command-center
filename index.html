Assignment Order — How Tickets Are Routed Read left to right per skill
When a ticket comes in, the system checks Tier 1 (Staff) first — routing to the qualified member with the fewest active tickets. If all Tier 1 members with that skill are at capacity, it escalates to Tier 2 (Director). If Tier 2 is also at capacity, it escalates to Tier 3 (VP/Executive). Property specializations always override tier order.
Define the skill categories used for auto-assignment. Each request type maps to one or more skills. The system routes to the qualified team member with the fewest active tickets.
